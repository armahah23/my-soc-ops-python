# 🎉 Welcome to Soc Ops!

A **Social Bingo Game** designed for in-person mixers! Break the ice, connect with others, and have fun finding people who match the questions to get 5 in a row! 🏆

---

## 🌟 Features

- **Interactive Gameplay**: Engage with others in a fun and dynamic way.
- **Customizable Questions**: Tailor the game to fit your event's theme.
- **Easy Setup**: Get started quickly with our step-by-step guide.
